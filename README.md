# JavaScript Arrays and Functions

## About this project

In this assignment, I practiced using arrays and functions in JavaScript.

I created functions that take an array as input and perform different operations like finding maximum value, calculating sum, and counting odd numbers.


## What I did

I used a sample array:

[4, 8, 2, 11, 6, 7, 10]

Then I created different functions to:

* Find the maximum number
* Find the sum of all elements
* Count how many numbers are odd

## Types of functions used

* Normal function - for maximum
* Arrow function - for sum
* Anonymous function - for counting odd numbers

## How to run

1. Open the JavaScript file
2. Run it in browser console or using HTML file
3. Check output in console

## What I learned

* How to loop through arrays using for loop
* How to compare values to find maximum
* How to use % operator to check odd numbers
* Difference between normal, arrow and anonymous functions


## Difficulties

* Initially confused between different function types
* Took time to understand how to update max value

## Conclusion

This assignment helped me understand arrays and functions better by actually writing logic using loops.
